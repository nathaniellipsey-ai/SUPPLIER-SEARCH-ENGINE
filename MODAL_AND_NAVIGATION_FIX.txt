================================================================================
  TECHNICAL DETAILS
================================================================================

MODAL STRUCTURE:
  - Fixed position overlay
  - Centered content area
  - Escape key support
  - Click outside to close
  - Scrollable if content is long

ADD SUPPLIER FLOW:
  1. User fills form
  2. submitNewSupplier() creates new supplier object
  3. Auto-generates missing fields (AI score, rating, etc.)
  4. Adds to allSuppliers array
  5. Updates filteredSuppliers
  6. Calls updateStats() and renderResults()
  7. Modal closes

IMPORT SUPPLIERS FLOW:
  1. User selects CSV file
  2. handleImportFile() reads file as text
  3. Parses CSV by splitting on newlines and commas
  4. Maps columns to supplier fields
  5. Creates supplier objects for each row
  6. Adds all to allSuppliers array
  7. Shows success message
  8. Calls updateStats() and renderResults()

NAVIGATION FLOW:
  1. User clicks user menu button
  2. toggleUserDropdown() creates dropdown menu
  3. Menu items navigate to respective pages
  4. Pages load independently
  5. Each page reads data from localStorage
