================================================================================
  WHY THIS WORKS
================================================================================

window.location.origin gives you:
  - http://localhost:3000 (when running locally)
  - https://supplier-search-engine.onrender.com (on Render)
  - https://yourserver.com (on any domain)

So the API calls automatically point to the right server!

API calls become:
  - http://localhost:3000/api/dashboard/suppliers (locally)
  - https://supplier-search-engine.onrender.com/api/dashboard/suppliers (on Render)

Your backend server.js serves BOTH the frontend AND the API!
